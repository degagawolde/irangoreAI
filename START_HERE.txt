🔧 ENVIRONMENT CONFIGURATION
═══════════════════════════════════════════════════════════════════════════

Key variables in .env:

  # Core
  DEBUG=false
  LOG_LEVEL=INFO

  # Neo4j Database
  NEO4J_URI=bolt://localhost:7687
  NEO4J_USERNAME=neo4j
  NEO4J_PASSWORD=password

  # LLM Provider
  LLM_PROVIDER=ollama              # or openai
  LLM_MODEL=qwen3:8b
  LLM_TEMPERATURE=0.7

  # Ollama (if using local models)
  OLLAMA_BASE_URL=http://localhost:11434

  # Sessions
  SESSION_TIMEOUT=3600
  MAX_HISTORY_LENGTH=50

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Neo4j Connection Failed?
  → Check NEO4J_URI in .env
  → Ensure Neo4j is running
  → Verify credentials

LLM Not Responding?
  → For Ollama: Run `ollama serve` in another terminal
  → For OpenAI: Set OPENAI_API_KEY
  → Check LLM_PROVIDER setting

Port 8000 in Use?
  → Kill existing process: lsof -i :8000 | grep LISTEN | awk '{print $2}' | xargs kill -9
  → Or use different port: fastapi dev main.py --port 8001

For more help:
  → See QUICKSTART.md for detailed troubleshooting
  → See README.md for comprehensive guide

│  │ load_config()          → Reads agents.yaml                          │  │

USAGE FLOW
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  1. Import                                                                   │
│     from agents import create_agent, generate_response                      │
│                         ↓                                                    │
│  2. Generate Response (Method A: Use default)                               │
│     response = generate_response("What's in the documents?")                │
│                         ↓                                                    │
│  3. Generate Response (Method B: Specify agent)                             │
│     response = generate_response("Question", agent_name="cypher")           │
│                         ↓                                                    │
│  4. AgentFactory                                                             │
│     • Load agents.yaml                                                      │
│     • Check cache for agent instance                                        │
│     • If not cached: create new agent                                       │
│     • Register tools from config                                            │
│                         ↓                                                    │
│  5. Execute Agent                                                            │
│     • Run LangChain ReactAgent                                              │
│     • Access Neo4j memory for session history                               │
│     • Return response                                                       │
│                         ↓                                                    │
│  6. Return Result                                                            │
│     response = "Answer based on documents..."                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

MEMORY & SESSION MANAGEMENT
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  Session Flow:                                                               │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ 1. User sends message with session_id                                 │ │
│  │ 2. Agent retrieved from cache or created                              │ │
│  │ 3. RunnableWithMessageHistory added                                   │ │
│  │ 4. Neo4jChatMessageHistory loaded for session                         │ │
│  │ 5. Agent processes with full conversation history                    │ │
│  │ 6. Response generated and stored in Neo4j                             │ │
│  │ 7. Next message has full context available                            │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│  Neo4j Storage:                                                              │
│     Session → ChatMessage history → Context for agent                       │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
